# `向量`使用教學文件

[toc]

`向量.js` 是一個擴展自 JavaScript 原生 `Array` 的模組，提供了一系列針對向量的專業化功能。以下以主題方式撰寫教學，涵蓋各向量操作的特性與使用方式。

---

## 1. **向量的構造與初始化**

### 建構函式
`向量` 的建構函式可以接收不同類型的參數，根據情況初始化向量。

#### 使用方法
```javascript
import 向量 from 'dx-core';
new 向量();                   // 創建長度為 0 的零向量
new 向量(5);                  // 創建長度為 5，元素皆為 0 的零向量
new 向量([1, 2, 3]);          // 初始化為 [1, 2, 3]
new 向量(1, 2, 3, [4, 5]);    // 初始化為 [1, 2, 3, 4, 5]
```

#### 注意事項
- 如果數字參數小於零，將拋出 `參數錯誤`。
- 如果傳入陣列長度為 1，例如 `[5]`，則向量會初始化為 `[5]`。

---

## 2. **向量基本屬性與操作**

### **範數**
計算向量的範數（歐幾里得長度）。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([3, 4]);
console.log(v.範數); // 5 (sqrt(3^2 + 4^2))
```

---

### **內積**
計算兩向量的內積值。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v1 = new 向量([1, 2, 3]);
const v2 = new 向量([4, 5, 6]);
console.log(v1.內積(v2)); // 1*4 + 2*5 + 3*6 = 32
```

#### 注意事項
- 若向量維度不同，將拋出 `參數錯誤`。

---

### **外積**
針對三維向量計算外積。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v1 = new 向量([1, 0, 0]);
const v2 = new 向量([0, 1, 0]);
console.log(v1.外積(v2)); // [0, 0, 1] (z軸)
```

#### 注意事項
- 僅支持三維向量，否則將拋出錯誤。

---

## 3. **向量運算功能**

### **加法**
可以將向量與另一個數、另一個向量相加。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v1 = new 向量([1, 2, 3]);
console.log(v1.加(1));         // [2, 3, 4]
console.log(v1.加(new 向量([4, 5, 6]))); // [5, 7, 9]
```

---

### **乘法**
將向量的每個元素乘以一個值。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([1, 2, 3]);
console.log(v.乘(2)); // [2, 4, 6]
```

---

### **加總**
計算所有向量元素的總和，可選擇排除某些元素。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([1, 2, 3, 4]);
console.log(v.加總()); // 10
```

---

### **交換**
交換向量中兩元素的位置。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([1, 2, 3]);
v.交換(0, 2);
console.log(v); // [3, 2, 1]
```

---

## 4. **向量查詢操作**

### **最大值**
查找向量中最大值的元素及其索引，可排除特定項目。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([1, 2, 3, 4]);
console.log(v.最大值()); // { 索引: 3, 值: 4 }
```

---

### **最小值**
查找向量中最小值的元素及其索引。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([1, 2, 3, -1]);
console.log(v.最小值()); // { 索引: 3, 值: -1 }
```

---

### **第一個大於**
找到第一個大於指定值的元素及其索引。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([1, 3, 5, 7]);
console.log(v.第一個大於(4)); // { 索引: 2, 值: 5 }
```

---

## 5. **向量的比較與驗證**

### **零向量檢查 (`是零向量`)**
判斷向量是否為零向量。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([0, 0, 0]);
console.log(v.是零向量()); // true
```

---

### **向量相等檢查 (`等於`)**
比較兩個向量是否相等。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v1 = new 向量([1, 2, 3]);
const v2 = new 向量([1, 2, 3]);
console.log(v1.等於(v2)); // true
```

---

## 6. **向量的遍歷與更新**

### **迭代**
針對向量的每個元素執行回呼函式。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([1, 2, 3]);
v.迭代((值, 索引) => console.log(值, 索引));
// 輸出:
// 1 0
// 2 1
// 3 2
```

---

### **更新全部**
根據回呼函式更新向量所有元素。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([1, 2, 3]);
v.更新全部(值 => 值 * 2);
console.log(v); // [2, 4, 6]
```

---

## 7. **向量的複製與擴展**

### **複製**
創建向量的深層複製。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([1, 2, 3]);
const newV = v.複製();
console.log(newV); // [1, 2, 3]
```

---

### **附加 **
將額外的元素添加到向量並返回新向量。

#### 使用方法
```javascript
import 向量 from 'dx-core';
const v = new 向量([1, 2, 3]);
const extendedV = v.附加(4, 5);
console.log(extendedV); // [1, 2, 3, 4, 5]
```

---

## 8. **錯誤處理相關功能**

- **索引超出範圍**：若操作的索引超出向量大小，將拋出錯誤訊息。
- **參數型別限制**：提供詳細的型別檢查，若型別不符，拋出相關錯誤。
