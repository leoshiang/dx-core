    位元陣列(initValue)

    位元陣列 適合用來記錄大量的 boolean 值，其記憶體使用量為用單一 boolean(4 bytes)的 1/24。

    new 位元陣列(initValue)
    位元陣列.js, line 15

    Parameters:
    Name Type Description
    initValue number | string 如果傳入數值，代表要存放幾個位元。如果isString，則會將字串視為二進位字串去解析。
    Throws:
    如果位元數不是大於零的數值會拋出此例外。
    Type
    型別錯誤
    Example
    let 位元陣列1 = new 位元陣列('01010101')
let 位元陣列2 = new 位元陣列(3)
